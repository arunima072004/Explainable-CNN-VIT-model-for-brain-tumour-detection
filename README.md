# Explainable CNN + ViT Model for Brain Tumour Detection

A full-stack AI web application that detects and classifies brain tumours from MRI scans using a two-stage pipeline — Vision Transformer (ViT) for detection and ResNet-50 for classification — with explainability visualisations and a downloadable PDF report.

---

## Features

- **Tumour Detection** — ViT-Base/16 binary classifier (tumour vs no tumour)
- **Tumour Classification** — ResNet-50 classifies Glioma, Meningioma, Pituitary
- **Explainable AI** — Grad-CAM heatmap, binary tumour mask, overlay visualisation
- **PDF Report** — Downloadable clinical-style report with all images and findings
- **Interactive UI** — Toggle between Original / Grad-CAM / Mask / Overlay views

## Setup & Running

### 1. Clone the repository

```bash
git clone https://github.com/Prashanth-Adibhatla/Explainable-CNN-VIT-model-for-brain-tumour-detection.git
cd Explainable-CNN-VIT-model-for-brain-tumour-detection
```

### 2. Add model weights

The trained model weights are not included in the repository (too large for GitHub).  
Place the following files inside the `backend/` folder:

```
backend/vit_binary_best.pth
backend/cnn_tumor_type_best.pth
```

These are generated by running the training cells in `project.ipynb` on Google Colab.  
To download from Colab after training:

```python
from google.colab import files
files.download('vit_binary_best.pth')
files.download('cnn_tumor_type_best.pth')
```

### 3. Backend setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

Backend runs at: `http://localhost:5000`

### 4. Frontend setup

Open a new terminal:

```bash
cd frontend
npm install
npm run dev
```

Frontend runs at: `http://localhost:3000`

## Usage

1. Open `http://localhost:3000` in your browser
2. Drag & drop or click to upload an MRI scan (JPG/PNG)
3. Adjust the masking threshold slider if needed (default 0.5)
4. Click **Analyze MRI**
5. View results — toggle between Original, Grad-CAM, Mask, and Overlay
6. Click **Download PDF** to get a full clinical-style diagnostic report

## API Endpoints

| Method | Endpoint   | Description |
|--------|-----------|-------------|
| GET    | `/health`  | Health check |
| POST   | `/predict` | Run inference on uploaded MRI image |
| POST   | `/report`  | Generate and download PDF report |

### POST `/predict`

Form-data:
- `image` — MRI image file (JPG/PNG)
- `threshold` — float 0.1–0.9, default 0.5

Response:
```json
{
  "tumor_detected": true,
  "confidence": 0.95,
  "tumor_type": "glioma_tumor",
  "type_confidence": 0.88,
  "all_probs": { "glioma_tumor": 0.88, "meningioma_tumor": 0.07, "pituitary_tumor": 0.05 },
  "bbox": [x, y, w, h],
  "heatmap": "data:image/jpeg;base64,...",
  "mask": "data:image/jpeg;base64,...",
  "overlay": "data:image/jpeg;base64,...",
  "original": "data:image/jpeg;base64,..."
}
```

## Model Architecture

| Stage | Model | Task |
|-------|-------|------|
| 1 | ViT-Base/16 (timm) | Binary: tumour vs no tumour |
| 2 | ResNet-50 (torchvision) | Multi-class: Glioma / Meningioma / Pituitary |

### Training (Google Colab)

The notebook `project.ipynb` contains the full training pipeline:
- **Cell 11** — Trains the ViT binary classifier (20 epochs, AdamW, CosineAnnealingLR)
- **Cell 13** — Trains the ResNet-50 classifier (30 epochs, AdamW, CosineAnnealingLR)

Dataset: [Brain Tumour MRI Dataset](https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset)  
Classes: `glioma_tumor`, `meningioma_tumor`, `pituitary_tumor`, `no_tumor`
